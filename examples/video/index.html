Tips for accessible Autoplay

  1. If you must have autoplaying ads on your web pages play it less than 5 seconds and then pause
  2. Provide a 'Pause', ‘Play’ button
  3. With videos avoid autoplay if you can and, if you must, code it so that hitting the ‘Esc’ key will stop it playing.
  4. Avoid combining audio with animations unless absolutely necessary.
  5. If the animation is not essential disable it when Reduce motion preference is enabled
