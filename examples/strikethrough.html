Naming of del and ins are not elements that you can consistently rely on to be named. For instance, JAWS with chromium/firefox browsers will not announce the aria-labels of these elements which is correct behavior. Similarly with iOS + VO. the del/ins elements are really meant to be used specifically in the context of marking up a document for edits - much like if using google docs. that said, nvda has implemented support for these so they will create announcements of "inserted" and "deleted"
